## 💻 설치 방법
1. 저장소 복제
```bash
git clone [레포지토리 URL]
cd [프로젝트명]
```

2. 가상환경 생성 및 활성화
```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
```

3. 의존성 설치
```bash
pip install -r requirements.txt
```

4. 애플리케이션 실행
```bash
python main.py
```
